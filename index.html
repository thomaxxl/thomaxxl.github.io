Linux Function Call Graph Visualizations and Coverage

Work in progress - incomplete !

Introduction

When browsing large codebases, it is often unclear how a certain function is reachable. I try to solve this problem for the linux kernel by generating call graphs. I then use dynamic instrumentation to generate code to reach different parts of the code.
